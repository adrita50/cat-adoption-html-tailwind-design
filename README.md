# cat-adoption-html-tailwind-design

This project is a web design based on a Canva template for a cat adoption social media post, recreated using HTML and Tailwind CSS. The design is fully responsive across different screen sizes.

## YouTube Video

Check out the YouTube video for the design tutorial: [Cat Adoption Design Tutorial](https://www.youtube.com/watch?v=pG3D6WH2INA)

## Features

- Fully responsive design for all screen sizes
- Clean and modern layout
- Cat adoption theme
